Additional blocks for the Plain Post Theme.

== Description ==

Plain Post Features is a companion plugin designed specifically for the Plain Post theme, a full site editing (FSE) supported WordPress theme. This plugin extends the theme's functionality by providing a collection of custom blocks tailored for creating dynamic and engaging content layouts using the WordPress block editor (Gutenberg).

With Plain Post Features, you'll have access to a diverse range of blocks optimized for the Plain Post theme, allowing you to effortlessly create stunning and responsive designs for your website.

== Installation ==

This section describes how to install the plugin and get it working.

e.g.

1. Upload the plugin files to the `/wp-content/plugins/plainpost-features` directory, or install the plugin through the WordPress plugins screen directly.
1. Activate the plugin through the 'Plugins' screen in WordPress


== Frequently Asked Questions ==

= What if I don't use the Plain Post theme? =

This plugin is specifically designed to complement the Plain Post theme. While the blocks may still work with other themes, full compatibility and optimized performance are guaranteed when used with the Plain Post theme.

= Can I contribute to this plugin? =

Absolutely! Contributions are welcomed. Please visit the GitHub repository for this project to contribute or report issues.
